# 인텔리제이 톰켓연동하는법

## 1. Tomcat(톰켓) 설치
1) 톰캣은 아파치 소프트웨어 재단에서 오픈 소스로 무료로 배포하는 서블릿 컨테이너이다.    

2) 다음 링크로 이동하여 설치 파일을 받아 보자.   
 - https://tomcat.apache.org/   
 - 원하는 버전을 선택 하도록 한다.   
   (나와 같은 경우는 Tomcat 9를 선택 하였다. Tomcat9는 Java 8부터 지원하니 참고 하자.)
